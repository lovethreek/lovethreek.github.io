在一台新的电脑写博客

使用github page+hexo免费搭建博客，是很多程序员的首选，国内的服务商都已经收费了，但是github从来没有收过费用，避免写着写着就打不开网站了，大家都很有默契选择这种方式。

和类似新浪博客这种平台式的博客不一样的地方，使用hexo需要安装以下软件：

  • git，根据自己的电脑配置，选择版本
  • node.js,同上

2.连接到 GitHub Pages 仓库

2.1. 生成 SSH 密钥

2.2. 将 SSH 密钥添加到 GitHub

  • 复制公钥内容：(刷黑之后，右键，点击copy)
  • 登录 GitHub，进入 Settings > SSH and GPG keys > New SSH key
  • 粘贴公钥并保存

2.3. 克隆你的博客仓库

这一步有其他的方式可以代替，比如说，直接用U盘拷贝源文件到新的电脑上。
